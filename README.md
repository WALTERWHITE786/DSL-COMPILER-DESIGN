## рҹ“Ң Project Description

A fully functional **End-to-End Compiler** for a custom Domain-Specific Language (DSL) built in C using **Flex (lexer)**, **Bison (parser)**, hand-written **AST**, **Semantic Analyser**, and an **LLVM IR Code Generator**. The generated `.ll` file is passed to `llc` for native machine code.

## вҡҷпёҸ DSL Language Features

```
# Comments start with #
program ProgramName {
    var x : int = 10;           # variable declaration
    var y : float = 3.14;
    var flag : bool = true;
    var msg : string = "hello";

    x = x + 5;                  # assignment

    if (x > 10) { ... } else { ... }   # if-else

    while (x < 100) { ... }    # while loop

    for (i = 0; i < 10; i = i+1) { ... }  # for loop

    print(x);                   # print
    input(x);                   # read input

    func add(a: int, b: int) -> int {   # function
        return a + b;
    }
} end
```

## рҹҡҖ Steps to Run

### Prerequisites
```bash
sudo apt install flex bison gcc llvm  # Debian/Ubuntu
```

### Build
```bash
make
```

### Compile a DSL file
```bash
./dslc test/sample1.dsl --ast --sym
```

### View generated IR
```bash
cat output/output.ll
```

### Run test suite
```bash
make test
```

### Full pipeline to executable (with LLVM)
```bash
./dslc test/sample1.dsl -o output/out.ll
llc output/out.ll -o output/out.s
gcc output/out.s -o output/program
./output/program
```
